## 前言

aoblog-node负责aoblog和aoblog-admin的后端部分。

## 技术

- cookie-parser: "~1.4.4"
- cors: "^2.8.5"
- debug: "~2.6.9"
- ejs: "~2.6.1"
- express: "~4.16.1"
- http-errors: "~1.6.3"
- jsonwebtoken: "^9.0.1"
- morgan: "~1.9.1"
- multer: "^1.4.5-lts.1"
- mysql: "^2.18.1"

## Build Setup ( 构建安装 )

```
# install dependencies
npm install

# serve with hot reload at localhost: 3001
npm start
```
